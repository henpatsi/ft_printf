# ft_printf

A partial recreation of the C library function printf.

Done as a project for my studies at Hive Helsinki.

#### Implemented conversions:
 - `%c` - character
 - `%s` - string
 - `%p` - pointer address
 - `%d` or `%i` - signed integer
 - `%u` - unsigned integer
 - `%x` - hexadecimal (lowercase)
 - `%X` - hexadecimal (uppercase)
 - `%%` - percent sign
